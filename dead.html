the perceptron was a music recommendation website written by maryrosecook.

A user typed in an artist they like and got recommended other, connected artists. Connections included friendship, music critic recommendation, a record release on the same label, co-occurrence on mixtapes and in people's music collections, and appearance at the same gig.

Each recommended artist had a description, sample track, links to the artist's Myspace and website, and a 'like' button.

When a user said they liked a recommended artist, listened to the artist's sample track or visited one of the artist's websites, the perceptron increased its trust in the connection's data source. Afterwards, recommendations from that source would be shown more prominently.

the perceptron is dead because I want to save $38 a month on hosting. I've backed up the database. The code is open source.
